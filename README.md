# Connor Burdick Recruiting & Portfolio Site

Production-ready Next.js 15 website for Connor Burdick, combining:
- NCAA soccer recruiting profile (Goalkeeper-first)
- Academic trajectory and GPA growth
- Software/AI builder portfolio

## Tech Stack
- Next.js 15 (App Router)
- TypeScript (strict)
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React
- shadcn/ui-style component patterns

## Run locally
```bash
npm install
npm run dev
```

## Validate
```bash
npm run lint
npm run build
```
